# AI Research Agent using n8n

## Overview
This project implements an AI research assistant capable of retrieving external information and generating summarized responses.

## Architecture
User Input → AI Agent → Web Search Tool → API → AI Summary

## Technologies
n8n
OpenAI
DuckDuckGo API

## Features
Tool-enabled AI agent
Dynamic web research
Automated summarization
API integration

## Workflow
The workflow is triggered by a user message and processed by an AI agent which decides whether to retrieve external information.

## Example Query
Research renewable energy

## Output
The system returns a summarized explanation of renewable energy and its main types.
